# projeto-gcs-trabalho3-grupo08

## Resumo

Trabalho prático requerido pela disciplina Gerencia de Configuração de Software, do curso Engenharia de Software UFG, 
e ministrada pelo professor Rubens de Castro Pereira

## Descrição

Este trabalho tem o objetivo de criar situações recorrentes da Gerência de Configuração, em um projeto de
Software. Este trabalho foi executado pelos alunos Bruno Messias, Nikolas Matias e Tiago Damascena, todos integrantes
do curso de Engenharia de Software.
